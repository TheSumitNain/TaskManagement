# Task Management Application

A user-friendly task management application built with **ReactJS** that allows users to create, organize, and manage their tasks efficiently. The app supports task creation, prioritization, status updates, sorting, filtering, and drag-and-drop reordering.

---

## Features

- **Task Creation**: Add new tasks with titles and manage their details.
- **Drag-and-Drop Reordering**: Easily reorder tasks using drag-and-drop functionality (powered by `react-beautiful-dnd`).
- **Task Status Management**: Mark tasks as completed or incomplete.
- **Prioritization**: Assign priority levels to tasks (High, Medium, Low).
- **Filters**: Filter tasks based on their completion status (All, Completed, Incomplete).
- **Sorting**: Sort tasks by creation time or priority.
- **Persistent Storage**: Saves tasks in `localStorage` so they persist across page reloads.
- **Navigation**: Navigate to task-specific details.

---

## How to use

- **Add a Task:**
  Enter a task title in the input field and click "Add Task."

- **Manage Tasks:**
  Use the drag-and-drop feature to reorder tasks.
  Click "Edit" to update a task title.
  Change the priority using the dropdown menu.
  Mark tasks as completed/incomplete with the toggle button.
  Delete tasks with the "Delete" button.

- **Filters:**
  Use the filter buttons to view all tasks, only completed tasks, or only incomplete tasks.

- **Sorting:**
  Use the "Sort By" dropdown to organize tasks by creation time or priority.

- **Navigation**: 
  Click on task name for task-specific details.
